Programming / Networking


Welcome to a hub of logic and connectivity. These programming and networking projects blend technical precision with creative problem-solving, shaping efficient systems and seamless integrations.

A snippet of code produced in the Arduino IDE meant for capturing data transmitted by a compatible magnetometer. The data would be displayed in the serial monitor on the 9600 hertz frequency channel and could be plotted to showcase fluctuations in surrounding magnetic fields.
The code, unfortunately, doesn't interpret data in a way that is understandable by humans. (Not yet)

I developed a text compression script using a well-known algorithm to compress game terrain data, which was then serialized with JSON for real-time map loading.
The compression also unintentionally introduced a layer of obfuscation, enhancing data security and making it more resistant to tampering.
This solution optimized performance while adding an extra layer of protection to the game’s map data.

I made a python script designed to intercept and log network traffic, providing insights into how data packets are transmitted and processed. The program was a test of real-time packet capture and analysis which was relatively straight forward (I developed severe paranoia as a result of this project). The program allows users to examine network protocols, decipher structured data, and understand the flow of communication between devices.
